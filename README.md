## 📖 About
A lightweight, terminal-based network packet analyzer that intercepts live traffic
and decodes protocol-layer details in real time. Built during the **CodeAlpha Cybersecurity Internship**.

## ✨ Features
- 📡 **Live Capture** — Continuously intercepts active network traffic streams
- 🔬 **Protocol Decoding** — Identifies TCP, UDP, and ICMP layer protocols
- ✂️ **Payload Truncation** — Decodes raw data into safe, human-readable ASCII snippets
- 💻 **Terminal-Based** — Zero GUI dependencies

## ⚡ Quick Start
```bash
git clone https://github.com/YourUsername/CodeAlpha_BasicNetworkSniffer
cd CodeAlpha_BasicNetworkSniffer
pip install scapy
sudo python3 sniffer.py
```
